Program Description: 
An application to pre-sell a number of cinema tickers. Each buyer can buy up to 4 tickets. No more than 20 tickets can be sold in total. Prompts the user for the desired number of tickets and then displays the number of remaining tickets after their pruchase. Application repeats until all tickets have been sold. Then displays the total number of buyers.

---

Logical Steps:  
1. Initialize Ticket Count and Buyer Count
2. Prompt for Ticket Purchase	
3. Check User input
4. Process the Ticket Purchase
5. Display Remaining Tickets
6. Check Remaining Tickets
7. Check for Tickets left
8. Display Total Number of Buyers

---

Variables:  
1. total_tickets: Total number of tickets available for sale. 
2. buyer_count: The number of buyers who have made a purchase.
3. requested_tickets: Number of tickets the user has purchased in a single transaction
4. max_tickets: Max number of tickets a buyer can purchase in one purchase
5. min_tickets: Minimum number of tickets a buyer must purchase in one purchase
